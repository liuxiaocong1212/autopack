1.list_grp为列表组的类，可以通过为其添加noborder类，控制list的上下border显隐

2.可以为li添加nav_item类，用来控制右侧箭头的显隐

3.icon_wrapper为左侧图标的容器，小图标，即字体图标，添加i标签

4.大图标即图片，添加带有.img_wrapper的div 和img标签

5.content类是文本域，title为文字,desc为描述，title必须保留，若文本为空，则标签内为空即可，若没有desc则不添加此标签即可
